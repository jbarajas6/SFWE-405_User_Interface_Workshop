# SFWE 405 – User Interface Workshop

## Overview

This project was created for the SFWE 405/505: Software Architecture and Design Course. The objective was to design and develop a clean, responsive, and visually appealing user interface using modern web development technologies.

The application serves as a personal portfolio webpage that introduces Jennifer Barajas Villarreal, highlights software engineering interests and technical skills, and provides professional contact information.

---

## Features

- Responsive single-page portfolio design
- Professional profile section with image
- Brief personal introduction
- Technical skills and interests display
- LinkedIn profile integration
- Email contact link
- Modern user interface built with reusable components
- Mobile-friendly layout

---

## Technologies Used

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Shadcn UI
- ESLint
- Prettier

## Installation

### Clone the Repository

```bash
git clone https://github.com/jbarajas6/SFWE-405_User_Interface_Workshop.git
```

### Navigate to the Project Directory

```bash
cd SFWE-405_User_Interface_Workshop
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:3000
```
